## 💡 WHEN YOU GET STUCK

### Stuck = Don't Give Up Yet!

**Try these IN ORDER:**

1. **Re-read the problem** (2 min)
   - Did you miss something?

2. **Plan on paper** (3 min)
   - Draw the state
   - List the functions

3. **Console.log everything** (3 min)
   - Log state
   - Log what's happening
   - Find the issue

4. **Try different approach** (5 min)
   - Maybe your way isn't working
   - Try simpler approach

5. **Google the error** (3 min)
   - Search exact error message
   - Not tutorials - just answers

6. **Take 5 min break** (5 min)
   - Walk, water, sit
   - Fresh eyes help

7. **THEN check solution** (5 min)
   - Not before this!
   - Now understand
   - Code again from memory
