## 💡 WHEN YOU GET STUCK

### Your Thought Process:
```
PANIC: "I don't know how to do this!"
BREATHE: "This is normal. Everyone feels this way."
THINK: "What does the problem ask? Break it down."
DEBUG: "console.log to see what's happening."
SEARCH: "Google this specific error."
PATIENCE: "Give it 30+ minutes first."
LEARN: "Then check solution to understand."
REMEMBER: "Code it again from memory."
SUCCESS: "You did it! You can do it again."
```

### Common Stuck Points:

**"I don't know where to start"**
→ Read problem again carefully  
→ Plan on paper first  
→ Don't code yet, just think

**"My code has errors"**
→ Read the error message carefully  
→ Use console.log to debug  
→ Google the error  
→ Try different approach

**"I understand the solution but couldn't code it myself"**
→ This is normal at first  
→ Code it again from memory tomorrow  
→ It'll be faster second time

**"This is too hard, can I skip?"**
→ No, but it's OK to take extra day  
→ Medium problems ARE harder  
→ That's why you're practicing
